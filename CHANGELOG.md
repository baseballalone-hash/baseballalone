## v0.8.0 ✓ — 성능 최적화 · 버그수정 · 리팩토링 (2026-06-09)

> **리팩토링 3원칙**: 1순위 성능 / 2순위 결합도 완화 / 3순위 중복 로직 제거

### 게임 디자인 변경

**능력치 간소화**
- `defense`(수비), `mental`(정신력) 2종 완전 제거
- 영향 범위: `player.js`, `npc.js`, `regression.js`, `simulator.js`, `milestones.js`, `awards.js`, `offseason.js`, `shopCatalog.js`

**재능 5종으로 통합** (기존 8종 → 5종)

| Key | 이름 | 가중 스탯 |
|---|---|---|
| `slugger` | 거포 | 파워 ×1.5, 컨택 ×1.3 |
| `sprinter` | 교타/주루 | 컨택 ×1.3, 주력 ×1.3, 선구 ×1.2 |
| `fireballer` | 강속구 | 구속 ×1.5, 스태미나 ×1.3 |
| `tactician` | 기교파 | 제구 ×1.5, 변화구 ×1.3 |
| `all_round` | 올라운더 | 전 스탯 ×1.1 |

**자동훈련 방향 7종으로 정리**: `slugger` / `contact_speed` / `batter_balance` / `fireballer` / `breaking` / `pitcher_balance` / `two_way`

**KBO 스탯 배율 교체**: `pro1` → `0.88`, `pro2` → `0.80`

### 성능 최적화

**NPC-only 경기 fast-path** (`880f961`):
- `week.js`: 플레이어 미참여 경기에 `{ isNpcOnly: true }` 전달
- `simulator.js`: `isNpcOnly` 시 타석 이벤트 전체 생략, OVR 기반 점수 직접 산출

**OVR 계산 중앙화** (`880f961`):
- 신설: `src/systems/ovrHelper.js` (`getBatterOVR` / `getPitcherOVR` / `getCombinedOVR`)
