# 9회말 환생 (Ninth Inning Rebirth)

주인공이 고교야구부 1학년(16세)부터 은퇴까지를 경험하는 웹 기반 커리어 시뮬레이션 게임.  
플레이어는 직접 경기를 조작하지 않고 **훈련 방향**과 **시간 흐름**만 통제하며, 시즌이 자동으로 진행된다.

야구 매니저 + 인생 시뮬레이션 + 자동 경기 진행의 결합체.

🌐 **배포 주소**: <https://baseball-alone.web.app>

---

## 게임 특징

| 항목 | 내용 |
|---|---|
| 시점 | 고교 1학년(16세) 입학 ~ 은퇴 |
| 포지션 | 양방향 (타자 + 투수 동시 능력치) |
| 조작 | 직접 컨트롤 없음. 훈련 방향만 정하고 시간을 흘려보냄 |
| 자동화 | 매 평일 자동 훈련, 매 주말 자동 경기 시뮬레이션 |
| 커리어 | 고교 → 대학 / KBO 드래프트 / MLB 오퍼 → 은퇴 → 회귀(NewGame+) |
| 언어 | 한국어 / English (상단 KO/EN 토글) |

## 빠른 시작

### 브라우저 플레이 (로컬)

```bash
# ES Module은 file:// 로 열면 CORS 에러 — 반드시 HTTP 서버 경유
python3 -m http.server 8765
# → http://localhost:8765 접속
```

### 시뮬레이션 검증 (Node ≥ 18)

```bash
node probe.mjs          # 단위 검증 — 마지막에 ✅ 전체 통과 확인
node probe-career.mjs   # 풀 커리어 22시즌 시뮬 — AVG .23~.30, ERA 2~6 확인
```

## 게임 흐름

1. **캐릭터 생성** — 이름 · 얼굴(6종) · 재능(5종, 회귀로 최대 3개) · 손잡이(4종)
2. **자동훈련 프리셋 선택** — 거포 / 교타주루 / 타자밸런스 / 강속구 / 변화구 / 투수밸런스 / 올밸런스
3. **재생 ▶** — 매 500ms마다 1일 진행 (속도 0.5x ~ 4x). 헤더의 ⏸/▶ 또는 ⚙ 설정 모달으로 제어
4. **평일** 자동 훈련 / **주말** 자동 경기 시뮬
5. **결승 진출 시** — 라이브 모달 (정면 POV + 다이아몬드 + 베지어 궤적)
6. **시즌 종료** — 자동 일시정지 → 휴식기 카테고리 선택 → 다음 학년/시즌
7. **졸업** — 대학 / KBO 드래프트 / MLB 오퍼 / 은퇴 진로 분기
8. **은퇴** — HoF 점수 산정 → 회귀 점수 적립 → 상점에서 영구 강화 → 다음 캐릭터 재시작

## 기술 스택

| 항목 | 결정 | 이유 |
|---|---|---|
| 플랫폼 | 웹 (Static HTML) | 링크만 공유하면 누구나 플레이. 빌드/설치 불필요 |
| 언어 | Vanilla JavaScript (ES Modules) | 의존성 0, 유지보수 부담 최소 |
| 그래픽 | SVG (코드 생성) + WebP 에셋 | 리소스 의존성 없이 동적 벡터 그래픽 |
| 저장 | localStorage + Firebase Firestore | 자동 세이브 + 클라우드 동기화 |
| 빌드 | **없음** | `index.html` 정적 서빙만으로 동작 |
| 다국어 | i18n 모듈 (KO/EN) | 키 기반 문자열 테이블, 토글로 즉시 전환 |
| 배포 | Firebase Hosting + GitHub Actions | `main` 푸시 시 자동 배포 |
| 광고 | GameMonetize SDK | 웹게임 전용 인터스티셜/보상형 광고 |
